# SDWU Beamer Theme
## 简介
 Shandong Women's University Beamer Theme
 因为学校一直没有latex版本的PPT，大家在正式场合的需要自己制作带有官方标识的PPT，本人因投简历需要，所以制作了一个采用官方标识标准的PPT供大家在学术汇报等正式场合使用。
 
 女院红(R:185 G:46 B:56)(C:30 M:100 Y:90 K:0)。
